Activation functions take a single numerical input and perform a certain mathematical operation on it

Step, switching from zero to one original version simulating transition from passive to active

Sigmoid, compressing between 0 and 1 continuously differentiable version of step function

Hyperbolic Tangent, floating from -1 to 1 like Sigmoid, but can also be negative

Relu: Rectified Linear Unit, negative values zeroed out, positive linear, fighting vanishing gradient
